# RULES — The Constitution

> エージェント（人間・AI問わず）がこのプロジェクトで絶対に守る憲法。
> ループの各 VERIFY 段階はこの RULES を基準に判定する。

## 第1条：尖りの原則（最優先・違反は即ボツ）
- 生成物が「競合の誰が書いても同じ」なら **FAIL**。平均・無難・出汁の薄い出力は失敗状態。
- 巨匠の思想は **混ぜない**。状況ごとに1つ（or 対立する2つ）を**選んで振り切る**。
- 迷ったら凡庸ではなく**極端**を選ぶ。その上で計測で検証する。

## 第2条：計測に繋がらないものは作らない
- すべての戦略・クリエイティブは「どの指標で成否を測るか」を**必ず**伴う。
- 測れないアウトプットは未完成とみなす。

## 第3条：役割分離（作る人 ≠ 採点する人）
- 生成エージェントと検証エージェントは必ず別。自己採点禁止。
- 「Boring Filter」エージェント（凡庸を殺す専任）の合格なしに出荷しない。

## 第4条：エンジニアリング・ガードレール
- シークレット（広告アカウントのトークン等）はコード直書き禁止。`.env` / secret manager。
- 外部APIは境界で必ず入力検証。
- 破壊的操作（実際の広告配信・課金発生）は**人間の承認ゲート**を必ず挟む。
- TDD: テスト→実装→green→コミット。

## 第5条：倫理・法務の下限
- 虚偽・誇大広告、各媒体のポリシー違反は生成しない。
- B2C 個人データはプライバシー法に従って扱う。
